## 功能简介：
- 商品浏览：商品的图片，售价，种类，简介以及库存等信息。
- 商品评论：用户在购买了商品后才会出现商品评论的文本框。
- 全文检索：支持对商品种类以及商品名称，简介的检索。
- 登录注册：用户的登录与注册，重置密码，密码发送到注册邮箱。
- 跳蚤使用协议：协议签属才能使用。
- 用户中心：支持用户个人信息，收货地址，修改密码，发布商品，修改个人信息，实名认证，等信息的更新，商品加入购物车，订单生成。
- 消息中心：支持用户回复商家信息。
- 卖家中心：在商品详细信息中通过联系卖家或则在评论区通过用户头像进入。
- 商品下单：在支付接口和企业资质的支持下可完成商品的下单功能，按照原子事务处理，下单异常则终止此次下单过程。
- 后台管理：支持后台管理功能，商品及用户信息的增加，更新与删除，可自定制样式与功能，日志，以及权限的管理和分配。
- 此项目基于django4.x开发

### 创建虚拟环境
最好是能够创建一个虚拟环境来运行项目
使用pycharm能很轻松的创建虚拟环境,这里不做介绍

### 电脑上如果没有mysql, 则需要安装mysql数据库
    解压 mysql-8.0.35-winx64.zip
	用管理员身份打开cmd命令窗口, 进入到刚刚解压的 mysql bin目录, 例如 cd D:\Program Files\mysql-8.0.35-winx64\bin
	依次执行以下命令
		mysqld --initialize --console  # 初始化数据库,初始化后会生成默认root密码,记得保存这个密码
		mysqld install  # 安装命令（安装mysql服务）
		sc config mysql start=auto  # 设置mysql开机自启
		net start mysql  # 启动mysql

		mysql -uroot -p  # 启动成功后, 用这个命令登录mysql, 此时会要求你输入mysql密码, 就是刚刚初始化数据库时生成的默认密码


	mysql安装启动成功后,需要修改下密码后才能正常使用
		alter user 'root'@'localhost' identified with mysql_native_password by '123123';  # 123123是你要设置的新密码


### 安装依赖包
`pip install -r .\requirements.txt`

### 创建mysql数据库
首先要确保机器里安装了mysql数据库, 为了方便可以使用 Navicat 工具连接数据库,创建名为 zaishichang 的数据库
将 zaishichang.sql 导入到数据库中
在django项目的配置文件 zaishichang/settings.py 里配置mysql信息
```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'zaishichang',
        'USER': 'root',  # 这里修改为你mysql数据库的用户名
        'PASSWORD': 'root',  # 这里修改为你mysql数据库的密码
        'HOST': 'localhost',
    }
}
```

### 创建超级用户(可选)
终端下执行:
<code>python manage.py createsuperuser</code>
然后输入相应的超级用户名以及密码，邮箱即可。

### 开始运行
终端下执行:
<code>python manage.py runserver 127.0.0.1:8080</code>
或者执行 start.py 脚本也可以

浏览器打开: <code>http://127.0.0.1:8080</code> 即可进入普通用户入口
浏览器打开: <code>http://127.0.0.1:8080/admin</code> 即可进入超级用户入口
